# lpe
LPE (Linux Privilege Escalation)

**Command:** * *python3 lpe.py* *

<b> Supports </b> python2 also.

<h3> What makes it special? </h3>
Finds vulnerabilities related to:

    1. GTFOBins

    2. sudo vulnerability
    
    3. SUID vulnerability
    
    4. capabilities
    
    5. cronjobs
    
    6. PATH
